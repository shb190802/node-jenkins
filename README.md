# node-jenkins
使用nodejs模拟的jenkins打包vue项目功能


基本流程：
```shell
> git clone https://github.com/shb190802/node-jenkins.git
> cd node-jenkins
> npm install
> node app.js
```

* 浏览器访问localhost:3011



可以将你的url发给局域网的小伙伴。
