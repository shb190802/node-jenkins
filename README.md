# node-jenkins
使用nodejs模拟的jenkins打包vue项目功能

**正式使用的时候可以使用koa创建一个webserver，封装API通过浏览器给团队其他人使用！**

基本流程：
```shell
> git clone https://github.com/shb190802/node-jenkins.git
> cd node-jenkins
> npm install
> node app.js
```

* 浏览器访问localhost:3011



可以将你的url发给局域网的小伙伴。
